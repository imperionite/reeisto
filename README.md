# REEISTO

**REEISTO (Rare Earth Elements Inventory System for Trading Operations)** is an internal operational platform that simulates inventory tracking and transaction workflows for rare earth materials trading. It models trader interactions with inventory across multiple warehouse locations, logging buy/sell intents as material allocations or acquisitions—without real financial trades
